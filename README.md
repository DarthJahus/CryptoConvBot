# CryptoConvBot
Simple cryptocurrency and fiat converter for Telegram and Discord.

❓ [Check the Wiki for detailed help](https://github.com/DarthJahus/CryptoConvBot/wiki)

* Discord commands: `cc/c [amount] <coin1> <coin2>`, `cc/p <coin>`, `cc/t <coin>`

* Telegram commands: `/convert [amount] <coin1> <coin2>`, `/price <coin>`, `/ticker <coin>`
